switch

使用

    1.在页面scss文件中引入下面这个文件
     @import "tobe/components/switch/switch";

    扩展）!如果以下颜色没有， 自行调用下面mixin, {$class}处命名改按钮颜色名称，$bgColor处填写背景颜色值
    .switch.switch-{$class} input:checked ~ span.bg{
      @include switch-color($bgColor);
    }

    2.html结构如下
    < 默认样式 >
    <label class="switch">
        <input type="checkbox" checked="checked">
        <span></span>
        <span class="bg"></span>
    </label>

    < 其他按钮样式 >
    <label class="switch switch-danger">
        <input type="checkbox" checked="checked">
        <span></span>
        <span class="bg"></span>
    </label>

widget

使用

gulp-file-include的配置已经写好了，详情见gulpfiles文件中 "widget" 这个task， 用法：首先svn更新，然后在根目录下执行npm install， 会自动安装package.json中配置的gulp-file-include插件包，安装完后执行gulp起服务，会自动检测app目录下的html变动自动执行widget任务。 

    扩展）!gulp-file-include插件的Github地址：https://github.com/coderhaoxin/gulp-file-include
    2.html结构如下
  基本用法
@@include('test.html')

@@include('some.html', { "nav": true })

@@if (name === 'test' && nav === true) {
  @@include('test.html')
}

例如：
包含公共头部，接收两个参数，第一个参数是title，第二个是要引入的css路径
@@include('widget/head.html', {
    "title": "选择配送方式",
    "css": "css/distribution.css"
})

header，参数比较多，如果页面头部不合适则不要使用
@@include('widget/header.html', {
    "left-icon": "arrow-back", //左边icon的类名
    "bar-style": "bar-dark", //bar-header的类名
    "title": "", //标题
    "nav": false, //是否引入j-nav
    "right-icon": "more-v", //右边icon的类名（rightIconBool参数一定要是true）
    "rightIconBool": true, // 是否显示右边icon
    "rightIconBoolMore": false, // 是否显示右边第二个icon
    "more-icon": "", //右边第二个icon类名
    "rightWordBool": true, //右边是否显示文字
    "right-word": "hello" //右边文字内容
})
