switch

使用

    1.在页面scss文件中引入下面这个文件 
     @import "tobe/components/switch/switch";

    扩展）!如果以下颜色没有， 自行调用下面mixin, {$class}处命名改按钮颜色名称，$bgColor处填写背景颜色值
    .switch.switch-{$class} input:checked ~ span.bg{
      @include switch-color($bgColor);
    }

    2.html结构如下
    < 默认样式 >
    <label class="switch">
        <input type="checkbox" checked="checked">
        <span></span>
        <span class="bg"></span>
    </label>

    < 其他按钮样式 >
    <label class="switch switch-danger">
        <input type="checkbox" checked="checked">
        <span></span>
        <span class="bg"></span>
    </label>

list-between

使用

    1.在页面scss文件中引入下面这个文件 
    @import "tobe/components/list-between/list-between";

    扩展）!下面预览除本组件还用到两个扩展（1.向右的灰色尖头，2.绿色选择按钮），扩展的样式不多，暂时全都放在ext/_comm.scss文件中，非输出型的，要用需要用@extend方法调用，选择按钮默认是绿色样式如需添加，自行调用 check-color($bgColor), $checkColor处填命名颜色名称，$bgColor处填写背景颜色
    /* 调用1.向右的灰色尖头方法 */
    .arrow-right{
        @extend %arrow-right;
    }
    /* 调用2.绿色选择按钮 */
    .arrow-right{
        @extend %arrow-right;
    }
    /* 调用其他颜色的选择按钮mixin  */
    .check-input.{$checkColor}:checked {
        @include check-color($bgColor);
    }

preloader

使用

    1.在页面scss文件中引入下面这个文件 
    @import "tobe/components/preloader/preloader";

    扩展）!默认有三个尺寸，分别分类了三个类，默认是最小的, 中等的加上.middle, 最大的加上.big, 颜色方面目前有下面两种，灰色和白色， 默认灰色， 白色需要加上white类
    2.html结构如下
    /* preloader */
    <div class="preloader"></div>
    <div class="preloader middle"></div>
    <div class="preloader big"></div>
    <div style="background: black;">
        <div class="preloader white"></div>
        <div class="preloader white middle"></div>
        <div class="preloader white big"></div>
    </div>

triangle css三角形

使用

    1.在页面scss文件中调用下面mixin,// 三个参数分别是，三角形方向，大小，颜色
    .triangle{
        @include triangle(left, 10px, red);
    }
    .triangleright{
        @include triangle(right, 10px, red);
    }
    .triangletop{
        @include triangle(top, 10px, red);
    }
    .trianglebottom{
        @include triangle(bottom, 10px, red);
    }

    扩展）!
    2.html结构如下
    <div class="triangle">
    <div class="trianglebottom"></div>
    <div class="triangleright"></div>
    <div class="triangletop"></div>
